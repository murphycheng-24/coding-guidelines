## [1.0.0] - 2026-07-07

### Added
- **10 engineering discipline rules** — enhanced from Karpathy's original 4 behavioral guidelines
  - Rule 1: Read Before You Write
  - Rule 2: Think Before You Code
  - Rule 3: Simplicity
  - Rule 4: Surgical Changes
  - Rule 5: Verification
  - Rule 6: Goal-Driven Execution
  - Rule 7: Debugging
  - Rule 8: Dependencies
  - Rule 9: Communication
  - Rule 10: Common Failure Modes
- **CLAUDE.md** — core guidelines file
- **SKILL.md** — reusable skill definition with frontmatter
- **.claude-plugin/plugin.json** — Claude Code plugin configuration
- **.cursor/rules/engineering-discipline.mdc** — Cursor IDE rule file
- **CURSOR.md** — Cursor setup guide
- **README.md** — English documentation with badges, install guide, and rule overview
- **README.zh.md** — Chinese documentation
- **LICENSE** — MIT license
- **.gitignore** — standard ignore patterns
- **Four-phase execution model** — Pre-flight, In-flight, Post-flight, Self-audit
- **Bilingual support** — English and 简体中文

### Changed
